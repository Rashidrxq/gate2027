# 🚀 Quick Deploy Commands for GATE 2027

## Deploy to Vercel (Frontend)
```bash
# Install Vercel CLI
npm install -g vercel

# Login to Vercel
vercel login

# Deploy frontend
cd frontend
vercel --prod
```

## Deploy to Render (Backend)
1. Push code to GitHub (already done ✅)
2. Go to render.com
3. Create new Web Service from GitHub repo
4. Select `render.yaml` configuration
5. Click Deploy

---

## Alternative: Deploy Both with One Command

### Using Vercel for everything (frontend + backend API routes)
```bash
# Install Vercel
npm i -g vercel

# Deploy
vercel --prod
```
